# Changelog

All notable changes to My Learning Jockey will be documented in this file.

## [Unreleased]

### Changed - 2026-01-13
- **大幅な文字数制限の拡大**: 書籍1冊分の内容に対応
  - タイトル: 200文字 → **500文字**
  - 内容: 5,000文字 → **500,000文字**（50万文字）
  - 書籍1冊分（約10〜20万文字）を余裕を持って入力可能

### Added - 2026-01-13
- 文字数カウンター表示の改善
  - 1,000文字区切りのフォーマット表示（例: 123,456文字）
  - 10万文字超過時に「≈ X冊分」の表示を追加
- Textareaのサイズ調整機能
  - 最小高さ: 300px
  - 最大高さ: 500px
  - リサイズ可能（`resize-y`）
- プレースホルダーテキストの改善
  - 「最大50万文字まで対応」を明記
  - 「書籍1冊分でも問題ありません」の案内追加

### UI Improvements - 2026-01-13
- ヒントセクションに追加情報
  - 📚 書籍1冊分（10〜20万文字）も対応の案内
  - 長文でも安心して貼り付けられることを強調
